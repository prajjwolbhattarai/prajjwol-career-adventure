# Prajjwol Bhattarai's Excellent Adventure – Career Edition

A Mario-style, pixel-themed gamified resume website showcasing Prajjwol Bhattarai's creativity, marketing leadership, and automation achievements.

---

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/prajjwol-career-adventure.git
cd prajjwol-career-adventure
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```

Visit `http://localhost:3000` to view your resume adventure.

---

## 🧱 Project Structure

```
/public
  ├── index.html          → Base HTML file
  ├── resume.pdf          → Downloadable resume
  ├── music.mp3           → Background music file

/src
  ├── main.jsx            → React entry point
  ├── App.jsx             → Root layout component
  ├── index.css           → Tailwind styles
  ├── /components         → All UI components
  ├── /data/worlds.js     → Career content per world
  └── /utils/GA4.js       → Google Analytics setup
```

---

## 🌍 Deployment with Vercel

1. Push to GitHub
2. Go to [vercel.com](https://vercel.com/)
3. Import GitHub project
4. Use default settings for Vite + React

---

## 🔧 Customization Tips
- Replace resume content in `worlds.js`
- Update social links in `SocialLinks.jsx`
- Drop your chatbot script in `ChatbotSlot.jsx`
- Add your GA4 ID in `GA4.js`
